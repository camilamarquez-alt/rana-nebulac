# Rana Nebulac

Juego web estático "Rana Nebulac" — una mascota pixel art interactiva. Este repositorio contiene index.html y los assets necesarios para ejecutar el juego.

URL pública (GitHub Pages): https://camilamarquez-alt.github.io/rana-nebulac/

Cómo usar:
1. Abre el enlace anterior o sirve los archivos localmente abriendo index.html en tu navegador.
2. Si no ves la rana, asegúrate de que rana-nebulac.png esté en la misma carpeta.
